# JungleQA

크래프톤 SW 12기 2반 2조의 프로젝트입니다. 

## 사용

### 실행 방법

```bash
docker compose up --build
```

입력한 터미널에서 맥이면 `cmd + c`, 윈도우면 `control + c` 눌러서 종료. 

### 접속

- 웹 앱: http://localhost:5001
- MongoDB: localhost:27017

### 종료

AI한테 도커 컴포즈 띄운거 삭제해달라고 ㄱㄱ
